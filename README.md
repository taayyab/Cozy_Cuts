# Cozy Cuts ✦ Beauty Salon Web App

A premium, highly animated frontend for a luxury beauty salon — built with **Next.js (App Router), React, Tailwind CSS, and GSAP**. Lenis powers buttery smooth scrolling.

## Features

- **Cinematic intro / loading screen** with logo reveal and curtain wipe
- **Animated navbar** with magnetic links and progress bar
- **Hero** with parallax, floating beauty icons, gradient orbs
- **Services** — animated 3D-tilt cards with on-scroll stagger reveals
- **Beauty Products Store** — featured carousel + category grid + hover lift
- **About / Team / Testimonials / Animated Counters / Contact / Footer**
- **Appointment Dashboard** (`/dashboard`) — sidebar, calendar, slot picker, staff selector, confirmation modal
- Glassmorphism cards, gold accents, blush + peach pastel gradients
- GSAP ScrollTrigger reveal animations everywhere
- Fully responsive (mobile / tablet / desktop)

## Quick start

```bash
npm install
npm run dev
# open http://localhost:3000
```

Visit `/dashboard` for the appointment management UI.

## Stack

- Next.js 14 (App Router, JSX)
- Tailwind CSS 3
- GSAP 3 + ScrollTrigger
- Lenis (smooth scroll)
- lucide-react (icons)
